Stretching Out
A Game about leaving a box

This project is my submission to the Bad Ideas Game Jam hosted by JonnyRazer
This project doesn't run any AI code*

My Socials:
Youtube - https://www.youtube.com/@moomoodylan

This reposatory contains all the source code and assets for the project.

Main Controls:

SPACE - Enable / Disable Legs

SCROLL WHEEL - Zoom in and out

A / D - Apply force in given direcion

W - Apply upward force

P - Show Debug
